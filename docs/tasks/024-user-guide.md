# Task #024: 公式编辑器操作指南

## Status: done

## 需求

面向非技术人员的 step-by-step 操作指南，覆盖公式图形编辑、文本编辑、LaTeX 预览等。中/英/日三语版本。

## 大纲

1. 概述 — 公式编辑器是什么，能做什么
2. 界面布局 — 顶栏、画布、侧边栏、测试面板
3. 节点类型一览 — 9 种节点的用途和图标
4. 基本操作
   - 创建节点（从调色板拖入画布）
   - 连接节点（拖动端口）
   - 编辑属性（右侧面板）
   - 删除节点/连线
   - 自动布局
5. 进阶功能
   - 子公式引用
   - 表查询
   - 条件分支
   - 循环（Loop）— sum/product/fold
6. 文本编辑模式
   - 文本语法说明
   - LaTeX 预览
   - LaTeX 输入模式
   - 文本↔图形切换
7. 测试和计算
   - 输入测试参数
   - 查看计算结果
8. 保存与版本管理
